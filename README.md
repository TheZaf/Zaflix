
# 🎬Zaflix a Movie & TV Show Web App

A responsive web application that allows users to explore movies and TV shows using data from the **TMDB (The Movie Database) API**. The app provides an interactive interface where users can browse trending content, view detailed information about movies and TV shows, and watch trailers.

## 🚀 Features

* 🎥 Browse movies by **Top Rated**, **Popular**, and **Now Playing**
* 📺 Explore **TV Shows** in a dedicated section
* 📄 **Detailed Movie/TV Pages** with overview, ratings, and release date
* ▶️ **Watch Trailers** directly within the application
* 🔐 **User Authentication** with **Signup and Login**
* 🍪 **JWT Authentication stored in Cookies** for secure session management
* ⚡ **Dynamic Data Fetching** from the TMDB API
* 📱 **Responsive UI** for a smooth experience across devices

## 🛠️ Tech Stack

**Frontend**

* HTML
* tailwindcsss
* React.js
* zustand (state management)
* axios (for fetching data)
* JavaScript

**Backend**

* Node.js
* Express.js

**Authentication**

* JWT (JSON Web Tokens)
* Cookie-based session handling

**API**

* TMDB API
